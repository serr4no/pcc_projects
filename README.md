# Python Crash Course Projects (3rd Edition)

This repository contains the three major projects from *Python Crash Course, 3rd edition*:

- **Alien Invasion**
- **Data Visualization**
- **Web Application**
 
A shared virtual environment is used for all projects, and each project has its own folder.

## Project Structure
```
pcc_projects/
│-- alien_invasion/
│-- data_visualization/
│-- web_app/
|-- sideways_shooter/
│-- .venv/ # ignored
│-- .gitignore
│-- .gitattributes
│-- requirements.txt
│-- README.md
```

## Getting Started

1. Activate the virtual environment:
    ```
    .\.venv\Scripts\activate
    ```

2. Install dependencies:
    ```
    pip install -r requirements.txt
    ```

Happy coding!
